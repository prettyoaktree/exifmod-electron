# GitHub settings (homebrew-exifmod)

Apply these on [github.com/prettyoaktree/homebrew-exifmod](https://github.com/prettyoaktree/homebrew-exifmod) — **Settings → Branches** (or **Rulesets**).

| Rule | Setting |
|------|---------|
| Branch | `main` |
| Require a pull request before merging | **On** |
| Required approvals | **≥ 1** |
| Require review from Code Owners | **On** (see root `CODEOWNERS`) |
| Include administrators | **On** |

Ensure `CODEOWNERS` is merged before strict enforcement so you can complete the bootstrap PR if needed.
