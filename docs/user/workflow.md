# Core workflow

1. **Open a folder** — you’ll see a list of supported image files.
2. **Select one or more files** in the list.
3. **Review and edit** metadata in the right-hand pane (use presets for camera, lens, film, author, and the other fields as needed).
4. **Preview EXIF changes** (optional) — if you like seeing the exact tags that would be written, use this before you commit.
5. **Write pending changes** — that applies your edits to the selected files.

## Formats

EXIFmod works with **JPEG**, **TIFF**, and common **RAW** files.

- **RAW:** metadata goes into an **XMP sidecar** next to each file (so your original is left alone in the usual RAW sense).
- **JPEG / TIFF:** metadata is written **into the file**. You can use an **optional backup copy** when the app offers it.
